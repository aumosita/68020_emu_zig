# 68000 vs 68020 차이 요약

## 핵심 차이

- 주소 공간: 68000은 24비트 버스 중심, 68020은 32비트 주소 사용
- 스택 포인터: 68020은 `USP/ISP/MSP` 분리
- 명령어 확장: 비트필드, `CALLM/RTM`, `RTD`, `CHK2/CMP2`, `MUL*.L/DIV*.L`, `PACK/UNPK`, `CAS/CAS2`
- 예외/인터럽트: 68020은 예외 프레임과 제어 흐름이 확장됨

## 이 프로젝트에서의 반영

- 32비트 메모리 접근
- 스택 뱅킹 및 IRQ 주입 API
- 68020 확장 명령 다수 실행 구현
- Line-A/Line-F, ILLEGAL, privilege violation 등 예외 경로 검증

## 아직 범위 밖인 항목

- FPU(68881/68882) 실행
- 캐시/파이프라인 하드웨어 모델
